使用hexo搭建你的第一个个人博客

  • hexo
  • node.js
  • 博客

Hello，今天来教大家怎么用hexo搭建你的个人博客

首先介绍下Hexo，Hexo基于Node.js，出自台湾大学生tommy351(大学就这么强，好厉害-.-！)，Hexo生成的静态网站可以放在任意空间上，例如常见的PHP、ASP空间、FTP服务器、百度BAE、新浪SAE等空间，可以说只要能用Web访问的就可以搭建起Hexo博客。然而我们今天要讲的是如何部署到github。(免费的谁不喜欢，逃)

一、准备工作

1、git                 git官网，打开比较慢，你懂的。

2、node.js          nodejs官网， 自行下载

3、github账号    github官网 ，自行注册

二、开始安装

1. 安装依赖包

$ npm install

2. 安装hexo

创建博客文件夹(如hexo)并右击，选择Git Hash Here，输入以下命令：

$ npm install hexo -g

初始化建立项目：

$ hexo init

3. 生成博客

$ hexo g

4. 打开服务器

$ hexo s

默认localhost:4000，在浏览器下输入此地址，本地博客搭建完成。

三、将Hexo博客部署到github

1. 前面已经注册了github账号(没注册的童鞋请自行面壁-.-)，创建repository并将其命名为账号名，比如我的Github账号是zywthj，那么我应该创建的Repository的名字是：zywthj.github.io。

2. 部署 (zywthj替换成自己的用户名)

打开hexo文件夹下的_config.yml配置文件，把最下面的一部分改成如下：

deploy:
  type : git
  repo : https://github.com/zywthj/zywthj.github.com.git
  branch : master

然后执行命令：

npm install hexo-deployer-git --save

最后执行如下命令：

$ hexo g
$ hexo d

或者写成：

$hexo d -g

即可成功部署。


有的童鞋没有配置过ssh，有时候repo设置为https地址不能连接本地，这时候可以使用ssh

把repo的值改为ssh地址(zywthj替换成自己的用户名)：

git@github.com:zywthj/zywthj.github.io.git

在命令行输入：

$ ssh-keygen -t rsa -C "317797468@qq.com"

（填写自己注册github的邮箱地址）

然后一路按回车，直到出现：

Your identification has been saved in /c/Users/xxxx_000/.ssh/id_rsa.#生成的密钥

Your public key has been saved in /c/Users/xxxx_000/.ssh/id_rsa.pub.#生成的公钥

然后将id_rsa.pub下的内容全部复制，粘贴到github：登录github系统；点击右上角账号头像的“▼”→Settings→SSH kyes→Add SSH key。

title：随便填，

Key：粘贴复制的id_rsa.pub下的内容

最后“Add Key”。

现在再执行：

$ hexo d -g

登陆hcjp6.xyz    查看博客，部署成功！
