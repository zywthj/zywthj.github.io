常用运算符的一些坑

  • javascript
  • 运算符

总结乘除加减以及逻辑运算的一些坑

一、乘法操作符

1、Infinity与0相乘，结果为NaN：

console.log(Infinity*0); //NaN

2、Infinity与非0相乘(包括它自己)，结果为Infinity或-Infinity：

console.log(Infinity*2);  //Infinity
console.log(Infinity*-5); //-Infinity
console.log(Infinity*Infinity) //Infinity

二、除法操作符

1、Infinity自除结果为NaN：

console.log(Infinity/Infinity); //NaN

2、0被0除结果为NaN：

console.log(0/0); //NaN

3、非0除0结果为Infinity或-Infinity：

console.log(Infinity/0); //Infinity
console.log(5/0); //Infinity
console.log(-5/0); //-Infinity

4、Infinity被任何非自身数除结果都为0：

console.log(0/Infinity); //0
console.log(2/Infinity); //0

三、求模

1、被除数无限大或者除数为0，结果都为NaN：

console.log(Infinity%0); //NaN
console.log(2%0); //NaN
console.log(Infinity%Infinity); //NaN
console.log(Infinity%-5); //NaN

2、被除数有限大，除数无限大，结果为被除数

console.log(2%Infinity); //2
console.log(-2%Infinity); //-2
console.log(0%Infinity); //0

四、加减法

1、Infinity加-infinity结果为NaN：

console.log(Infinity-Infinity); //NaN

2、每个加法操作都是独立的，如下：

var n1 = 1;
var n2 = 2;
var str = "The sum is" + n1 + n2;
console.log(str); //The sum is 12; 
var str = "The sum is" + (n1 + n2);
console.log(str); //The sum is 3;

3、只有-0加-0的结果为-0，其余都为+0：

console.log(+0 + -0); //+0
console.log(+0 - -0); //+0
console.log(-0 - -0); //+0
console.log(-0 + -0); //-0

PS：NaN与任何操作数做运算结果都为NaN

五、关系运算符

如果两个操作数都是字符串，则比较两个字符串对应的字符编码值：

var result = "anmy" > "Bye"; //a的字符编码为97，B为66。
console.log(result); //true 
var result2 = "23" < "3"; //"2"的字符编码是50，而"3"的字符编码为51。
console.log(result2); //true
